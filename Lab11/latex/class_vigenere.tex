\doxysubsection{Detailed Description}
A vigenere cipher that both encrypts and decrypts using a set key. 

This class takes a key in its constructor that it will use to encrypt and decrypt messages. It can also change its key later using the set\+Key command. It can also tell if a pair of messages are the same message after being encrypted/decrypted using a vigenere cipher with the current key.
