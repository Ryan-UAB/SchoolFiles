\doxysubsection{Detailed Description}
A \doxylink{class_hamming}{Hamming} Code (7,4) class that can both decode and encode. 

This class takes a filename in its constructor where the file is a list of \doxylink{class_hamming}{Hamming} Codes for every line to be either decoded or encoded. If the given \doxylink{class_hamming}{Hamming} Code has an error, then it will be corrected. The \doxylink{class_hamming}{Hamming} Code pattern followed is d1d2d3d4p1p2p3.
